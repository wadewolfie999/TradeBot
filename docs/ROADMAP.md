# TradeBot Roadmap

## Current Execution Constraint

- Web, API, package registry, and GitHub-dependent tasks should be grouped into planned VPN/global-access windows.
- Offline design, local coding, local testing, and doc maintenance should remain the default between sync windows.
- Sync-window planning should cover dependency downloads, GitHub push/pull, Codex or other cloud-assisted work, and any exchange or external data checks that require global access.
