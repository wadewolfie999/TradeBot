# TradeBot Decisions

This directory contains permanent ADR-style project decisions.

Each decision records a project-level choice that should remain stable unless superseded by a later decision.

Execution history belongs in Git commits, pull requests, issues, and local Git history before sync.

The old MOP/MOR/SS workflow is deprecated and is not part of the active TradeBot workflow.

## Index

- `0001-deprecate-offline-mop-mor-ss.md` — Deprecate the old offline/intranet-era MOP/MOR/SS workflow.
- `0002-github-as-system-of-record.md` — Establish GitHub as the system of record after this documentation cleanup is committed and pushed.
