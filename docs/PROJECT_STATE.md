# TradeBot Project State

## Current Operating Condition

- Development is running under intermittent or intranet-like global connectivity from Iran.
- Premium VPN bandwidth is limited and should be treated as a scheduled resource.
- Default operating mode is offline-first and low-bandwidth-conscious.
- GitHub remains the long-term system of record, but local Git is the immediate working record until the next VPN/global sync window.
- Dependency installs, cloud/Codex usage, web lookups, exchange access, and other external API work should be deferred into planned connectivity windows where practical.
